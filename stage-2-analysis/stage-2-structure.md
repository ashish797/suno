## PART B: SONG STRUCTURE TYPES

### Type 1: Standard 2-Verse-2-Chorus (10/20 songs = 50%)
```
[Verse 1] → [Chorus 1] → [Verse 2] → [Chorus 2] → [Bridge?] → [Chorus 3]
```
| Songs | Notes |
|-------|-------|
| HUSN | No bridge, chorus IS the hook |
| KASOOR | Bridge present, 3× chorus |
| TU KAHAN | Short verses, conversational |
| DOORIYAN | Male/female duet structure |
| RAASTA | Verse opens with question |
| TERE HI HUM | Minimalist — 4-line verse, 3-line chorus |
| KUCH DIN | Near-identical chorus repeats |
| RATEIN GUZAARI | Confrontational verse, intimate chorus |

| TINKA | Chorus-dominant, minimal verses |

---

### Type 2: Verse + Full-Chorus-as-Hook (5/20 songs = 25%)
```
[Verse] → [Hook Chorus — full 4-8 lines] → [Verse] → [Hook Chorus]
```
| Songs | Notes |
|-------|-------|
| JO TUM MERE HO | Very short lines (3-6 words), chorus lands fast |
| KHO GAYE HUM KAHAN | Imagery-heavy verses, simple chorus repeats |
| GUL | Poetic verses, title-word chorus anchor |
| KUCH DIN | Minimal variation between chorus repeats |
| UNSTOPPABLE | Word repeated 8x in hook |

**Finding:** These songs DON'T have a tight 2-line hook. Instead they use a **full 4-8 line chorus** as the hook unit. The entire chorus IS the hook.

## PART E: VERSE STRUCTURE PATTERNS

### E1. Verse Opening Lines — What Gets Attention First?
| Song | Opening Technique |
|------|------------------|
| TU KAHAN | Question — pulls listener in |
| TU HAI KAHAN | Question — "Tu hai kahan?" |
| RAASTA | Question — "Raato ki tanahaiyaan..." (lament) |
| ALAG AASMAAN | Statement — "Nayi nahi hain, yeh baatein vahi..." |
| KASOOR | Narration — first person, observational |
| HUSN | Request — "Aankhon mein toh dekho" |
| TINKA | Sensory — "Teri yaadon ki baarish mein..." |

| UNSTOPPABLE | Defeat statement — "Life se aaya tha vo tang..." |

- **Questions** pull listener in (TU KAHAN, TU HAI KAHAN, RAASTA)

- **Requests/demands** create intimacy ("Dekho na", "Aaye jaana")
- **First-person confession** creates vulnerability

## PART H: KEY STRUCTURAL RULES FOR SUNO

### Hook Timing
- ✅ Hook within 30 seconds (65% of songs)
- ⚠️ 45-60 seconds is risky — only works if the build-up earns it
- ❌ Never delay hook past 60 seconds

### Hook Size
- ✅ 60% use full 4-6 line chorus as the hook unit — NOT a short tagline
- ⚠️ 2-line hooks work (KASOOR, TERE HI HUM) but need to be extremely tight
- ❌ Single word hooks only work if repeated 6-8× (UNSTOPPABLE)

### Repetition
- ✅ Hook repeats minimum 3× per song
- ✅ End the song on the hook, not a verse

### Verse
- ✅ Keep verses tight: 4-6 lines max
- ✅ First line should either ask a question OR make a bold statement

### Structure
- ✅ Standard: Verse → Chorus → Verse → Chorus → Bridge → Chorus
- ✅ For minimalist style: Verse (2-4 lines) → Chorus → Verse → Chorus → Chorus

### Runtime
- ✅ Target 3:00-3:30 (75% of songs fall here)
- ❌ Avoid going over 4:00 for streaming + Reels use case

### Duet Structure
- ✅ Dino James pattern: male rap verses + female sung chorus = strong contrast
- Works well for heartbreak/longing songs where the other person's voice is the "desired" presence
