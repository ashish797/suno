## 🎵 WHAT WE KNOW SO FAR

### What makes Hindi indie songs go viral:
1. **One-line quotable hook** — lands in first 30 seconds
2. **Length 2:15 - 2:45** — Reels-friendly
3. **Late-night emotional resonance** — 2 AM heartbreak is the core driver
4. **Lo-fi indie production** — clean but not overproduced
5. **Hindi pronunciation clarity** — must be understandable
6. **Self-worth / heartbreak themes** — strongest emotional categories
7. **Social-first design** — built for the 15-second Reels test

### The platform virality path:
```
YouTube Shorts / Instagram Reels →
Spotify search →
Release Radar fires →
Playlist discovery →
JioSaavn picks up →
Cross-platform growth
```

### Current season (April 2026):
Late spring → summer incoming. Good windows for:
- Travel / freedom songs
- Pre-monsoon anticipation (romance building)
- Fresh heartbreak angles (post-valentine's new lens)
